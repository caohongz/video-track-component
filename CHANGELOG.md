# 更新日志

## [0.1.0] - 2025-01-01

### 新增

- ✨ 初始版本发布
- ✨ 支持多种轨道类型（视频、音频、字幕、贴纸、滤镜、特效、转场）
- ✨ 完整的片段编辑功能（拖拽、调整时长、多选）
- ✨ 时间线控制（动态刻度、卡尺拖拽、吸附）
- ✨ 播放控制（播放/暂停、速率调整）
- ✨ 撤销/重做功能
- ✨ 完整的快捷键系统
- ✨ 右键菜单
- ✨ 框选与多选
- ✨ 主轨道模式
- ✨ 自定义扩展支持
- ✨ 性能优化（虚拟滚动、懒加载）
- ✨ 完整的事件回调系统

### 特性

- 🎨 现代化的 UI 设计
- 🚀 基于 Vue 3 + TypeScript
- 📦 使用 Pinia 进行状态管理
- ⚡ Vite 快速构建
- 🎯 完整的 TypeScript 类型支持
- 🧩 组件化架构，易于扩展
- 💪 性能优化，支持大量轨道和片段
