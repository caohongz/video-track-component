# 样式设计指南

## 设计理念

音视频轨道组件采用现代化、充满活力的视觉设计，强调简洁明了和美观大方。

## 配色方案

### 主色调

- **渐变蓝紫色系**：`#667eea` → `#764ba2`
- 用于主要交互元素、高亮状态、品牌标识

### 辅助色

- **成功色**（绿色）：`#10b981` - 用于音频轨道
- **警告色**（橙色）：`#f59e0b` - 用于字幕、滤镜轨道
- **危险色**（红色）：`#ef4444` - 用于删除等危险操作
- **信息色**（蓝色）：`#3b82f6` - 用于提示信息

### 背景色

- **深色背景**：`#0f0f23` - 应用主背景
- **中等背景**：`#1a1a2e` - 组件容器
- **浅色背景**：`#16213e` - 工具栏、轨道
- **更浅背景**：`#1f2937` - 按钮、输入框

### 边框色

- **主边框**：`#374151`
- **浅边框**：`#4b5563`

### 文字色

- **主文字**：`#f3f4f6`
- **次文字**：`#9ca3af`
- **辅文字**：`#6b7280`

## 视觉特效

### 阴影系统

- **小阴影**（sm）：用于卡片、按钮
- **中等阴影**（md）：用于悬浮面板
- **大阴影**（lg）：用于弹窗、菜单
- **发光阴影**（glow）：用于高亮元素

### 圆角规范

- **小圆角**（sm）：4px - 用于小元素
- **中等圆角**（md）：6px - 用于按钮
- **大圆角**（lg）：8px - 用于卡片
- **超大圆角**（xl）：12px - 用于容器

### 过渡动画

- **快速过渡**（150ms）：按钮、开关
- **标准过渡**（250ms）：卡片、面板
- **慢速过渡**（350ms）：大型元素

## 组件样式

### 工具栏（ToolsBar）

- 采用毛玻璃效果（backdrop-filter）
- 按钮支持渐变悬浮效果
- 播放按钮采用圆形渐变设计
- 缩放滑块使用渐变滑块

### 时间线（Ruler）

- 游标采用渐变色 + 发光效果
- 游标手柄支持悬停放大动画
- 刻度标签清晰易读

### 轨道组（Tracks）

- 轨道悬停时有微妙的背景高亮
- 锁定轨道使用斜纹背景
- 添加按钮支持悬停动画

### 轨道控制栏（TrackControl）

- 主轨道徽章采用渐变设计
- 控制按钮悬停时上移 + 发光
- 名称编辑时有聚焦发光效果

### 片段（Clip）

- 所有 Clip 采用圆角卡片设计
- 悬停时上移 + 边框高亮 + 阴影增强
- 选中状态有明显的外发光效果
- 调整手柄仅在悬停时显示
- 渐变背景：
  - 视频：蓝紫渐变 `#667eea → #764ba2`
  - 音频：绿色渐变 `#10b981 → #059669`
  - 字幕：橙色渐变 `#f59e0b → #d97706`
  - 贴纸：粉色渐变 `#ec4899 → #be185d`
  - 滤镜：橙色渐变 `#f59e0b → #d97706`
  - 特效：紫色渐变 `#8b5cf6 → #7c3aed`
  - 转场：青色渐变 `#06b6d4 → #0891b2`

### 右键菜单（ContextMenu）

- 毛玻璃背景 + 大阴影
- 菜单项悬停时渐变高亮
- 打开时有淡入 + 缩放动画

## 微动画

### Clip 图标动画

- **贴纸**：闪烁动画（sparkle）- 缩放 + 旋转
- **滤镜**：浮动动画（float）- 上下移动
- **特效**：旋转动画（rotate）- 360° 旋转
- **转场**：滑动动画（slide）- 左右移动

### 游标动画

- 脉冲动画（pulse）- 透明度变化

### 悬停反馈

- 所有可交互元素支持 `transform: translateY(-1px)`
- 按钮悬停时有微妙的发光效果

## CSS 变量使用

所有样式使用 CSS 变量（CSS Custom Properties），便于主题切换和维护：

```css
/* 使用示例 */
.element {
  color: var(--color-text-primary);
  background: var(--color-bg-light);
  border-radius: var(--radius-md);
  transition: all var(--transition-fast);
}
```

## 响应式设计

- 使用 `flex` 和 `grid` 布局
- 轨道宽度自适应
- 工具栏按钮自动换行
- 支持横向和纵向滚动

## 无障碍设计

- 保持足够的颜色对比度
- 悬停状态清晰可见
- 禁用状态明确标识
- 聚焦状态有视觉反馈

## 性能考虑

- 使用 `transform` 和 `opacity` 实现动画（GPU 加速）
- 避免频繁的重排和重绘
- 使用 `will-change` 优化动画性能
- 合理使用 `backdrop-filter`（可能影响性能）

## 浏览器兼容性

- 使用 `-webkit-` 前缀支持 Safari
- 渐变、阴影等特性在现代浏览器中全部支持
- `backdrop-filter` 在旧版浏览器中会优雅降级
